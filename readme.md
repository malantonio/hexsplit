# hexsplit

splits a hexidecimal string into its hex character pairs.

## example
```javascript
> var h = require('hexsplit');
undefined
> h('5f4af7e27a2e7c4f5099d3edcda126f0');
[ '5f',
  '4a',
  'f7',
  'e2',
  '7a',
  '2e',
  '7c',
  '4f',
  '50',
  '99',
  'd3',
  'ed',
  'cd',
  'a1',
  '26',
  'f0' ]
```
